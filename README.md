# Meritts API Test

Repositório destinado para teste técnico na empresa Meritts.

## Instalação

```bash
$ npm install
```

## Rodando a API

```bash
$ npm run start
```

## Provas (Exams)

### Recuperando todas as provas

| Tipo | Rota | Parâmetros | Retorno |
|------|------|------------|---------|
|GET   | /exams | none | Exams[] |
